Definindo grid-areas:
-   Vamos supor que eu queria criar uma caixa que ocupe metade do meu grid e, de forma simplificada, posso fazê-la assim:

.caixa {
    grid-row-start: 1;

    grid-column-start: 1;

    grid-row-end: span 3; -> ou seja, usando span, você indica a quantidade que foi expandida baseada no row-start, então se começa no row-start: 1, e tendo um span na row-end de 3, seria o mesmo que dizer que o row-end teria 4 (1 + 3 de expansão = 4), o mesmo para colunas

    grid-column-end: span 2; -> o mesmo que dizer que o column-end está com 3, mas nesse caso estou usando span para indicar a quantidade de expansão baseada na column-start (1 + 2 de expansão = 3)
}

Simplificando declarações (grid-row, grid-column):

ao invés de fazer:
    .caixa {
        grid-row-start: 1;
        grid-column-start: 1;
        grid-row-end: 4;
        grid-column-end: 3;
    }

Eu posso fazer:
    .caixa {
        grid-row: 1 / 4 ou span 3; -> ou seja, start: 1 & end: 4

        grid-column: 1 / 3 ou span 2 -> column-start: 1 & column-end: 3
    }

Também posso simplificar ainda mais, usando:
    grid-area: 1 (row-start) / 1 (column-start) / 4 (row-end) / 3 (column-end);

    ou seja: grid-area: 1 / 1 / 4 / 3;

EM RESUMO:

    .caixa {

    }

    .caixa {
        grid-row: 1 / 4 ou span 3; -> 

        grid-column: 1 / 3 ou span 2 
    }

Forma simplificada 2:
    .caixa {
        grid-area: 1 / 1 / 4 / 3;

Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag </b> (NÃO SEMÂNTICA)

Nesta outra frase, temos um termo em negrito usando a tag </strong> (SEMÂNTICA)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag </i> (NÃO SEMÂNTICA)

Nesta outra frase, temos um termo em ênfase usando a tag </EM> (SEMÂNTICA)

Texto marcado

Podemos criar também um texto marcado usando a tag </MARK> (SEMÂNTICA)

Texto deletado

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado, isso com a tag </del> (SEMÂNTICA)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag </ins> (SEMÂNTICA)

Existe também o sublinhado com a tag </U> (NÃO SEMÂNTICA)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag </sup> (SEMÂNTICA)

Texto subscrito

Para inserir coisas do tipo H20 usando a tag </sub> (SEMÂNTICA)
